funcLib Apps Readme for 1.0.1

- Everything listed here started as a demo for funcLib but advanced to become much more. These are full and
  complete apps to be run on your PSP. You are now free to browse through the source of these programs. 

  - flAstro: A great game to test out funcLib's capabilities, later released to the public as a fun little game.
  - flAstro4MHz: flAstro minus some HUD features and with 16 bit framebuffers, making it faster. Runs only at 4MHz.
  - flMineSweeper: A full game of MineSweeper without a menu. 
  - flPassword: Used to test funcLib, lets you create a password and saves it to a text file which it will use to 
                reference the genuinity of your entered password.
