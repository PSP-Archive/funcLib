funcLib Documentation Readme for 1.0.1

- The latex folder is for printing the documentation, a .tex is like a .pdf or .html in that it's readable by
  humans and will print accordingly.

- The html folder is for browsing through the normal documentation files, there is a shortcut to the index which
  you should follow if you want to read up on funcLib. 

- The funclib.cfg file is for generating documentation via doxygen.
